# Image_inpainting

## Usage

**Python**

`python3 inpaint.py -i sample.jpeg`

**C++**

```cpp
g++ inpaint.cpp `pkg-config opencv --cflags --libs` -o inpaint
./inpaint sample.jpeg    
```
